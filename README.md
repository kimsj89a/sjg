# WORKFLOW Manager

A lightweight, browser-based Gantt Chart tool for managing project workflows and notes.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## Features

### Project Management
- Create, edit, and delete multiple projects
- Switch between projects via sidebar
- Project descriptions for additional context

### Gantt Chart
- Visual timeline for task scheduling
- Set start/end dates for each task
- Track progress (0-100%) with visual indicators
- 10 color options for task categorization
- Zoom in/out for timeline adjustment
- Weekend and today highlighting

### Memo System
- Project-specific notes
- Create, edit, and delete memos
- Timestamped entries

### Data Persistence
- Automatic LocalStorage saving
- No server required
- Data persists across browser sessions

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/kimsj89a/sjg.git
```

2. Open `index.html` in your browser

That's it! No build process or dependencies required.

## Usage

1. **Create a Project**: Click "+ 새 프로젝트" in the sidebar
2. **Add Tasks**: Click "+ 태스크 추가" to add workflow items
3. **Set Dates**: Define start and end dates for each task
4. **Track Progress**: Update progress percentage as work completes
5. **Add Notes**: Click "메모" button to open the memo panel
6. **Zoom**: Use +/- buttons to adjust timeline scale
